# ft_turing

School project aiming to learn Turing Machine and functionnal programmation.

Coded in Haskell

### Dependency

- Cabal
- Haskell2010
- ormolu

### Run

```shell
    cabal build
    cabal run
```

### Contributing

Install `ormolu`

```shell
    cabal install ormolu
```

Install `pre-commit` hooks

```shell
    pip install pre-commit
    pre-commit install
```
