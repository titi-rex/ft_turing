# ft_turing

School project aiming to learn Turing Machine and functionnal programmation.

Coded in Haskell

## Install

### Dependency

- Cabal
- Haskell2010
- ormolu

### Run

Build the package

```shell
    cabal build
```

Run directly

```shell
    cabal run exes [--] [FLAGS] [ARGS]
```

install locally

```shell
  cabal install --installdir=.
```

### Contributing

Install `ormolu`

```shell
    cabal install ormolu
```

Install `pre-commit` hooks

```shell
    pip install pre-commit
    pre-commit install
```
